Usuarios da API WEB

JSONPlaceholder é uma API REST online gratuita que você pode usar sempre que precisar de alguns dados falsos. É ótimo para tutoriais, teste de novas bibliotecas, compartilhamento de exemplos de código, ...


Se você clicar em usuários, será direcionado para a lista disponível de usuários da API


Se você clicar em posts, será direcionado para a lista disponível de posts da API


JSONPlaceholder vem com um conjunto de 6 recursos comuns:

  • users 10 users
  • posts 100 posts
  • comments 500 comments
  • albums 100 albums
  • photos 5000 photos
  • todos 200 todos

Para este exercício, componentes e serviços são criados para habilitar as APIs em nosso aplicativo. Foram definidas as rotas e a integração lógica das mesmas.
